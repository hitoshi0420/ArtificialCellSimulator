# Artificial Cell Simulator

A simulation tool that models basic cell behaviors and interactions, demonstrating emergent properties of cellular systems.

## Features
- Simulate basic cell life cycle (birth, growth, division, death)
- Cell interaction and signaling
- Visual rendering of cell populations
- Configurable simulation parameters

## Tech Stack
- Python

## Quick Start
`ash
pip install -r requirements.txt
python main.py
`
